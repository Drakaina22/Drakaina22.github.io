Code Log 1: HTML Semantic Tags

Questions

What is the difference between non-semantic and semantic tags in HTML?

  • Semantic tags tell what kind of content is inside. Non-semantic tags do not describe the content's role or structure.

What are two (or more) benefits of using semantic tags?

  • One benifit of using semantic tags is accessibility. It's easier for people with disabilities to navigate a website with semantic HTML. Another benifit is browser compatibility, HTML semantic tags also make sure that code can be integrated with future technologies.
